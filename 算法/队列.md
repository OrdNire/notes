# 队列

**队列只在队列的首部(head)处进行删除操作，在尾部(tail)处进行插入操作，队列中无元素时(head=tail)称为空队列**

例子：

**解密QQ号**

```
新学期开始了，小哈是小哼的新同，小哼向小哈询问QQ号，小哈当然不会直接告诉小哼。所以小哈给了小哼一串加密过的数字，同时小哈也告诉了小哼解密规则。规则是这样的：首先将第1个数删除，紧接着将第2个数放到这串数的末尾，再将第3个数删除并将第4个数再放到这串数的末尾，再将第5个数删除……直到剩下最后一个数，将最后一个数也删除。按照刚才删除的顺序，把这些删除的数连在一起就是小哈的QQ啦。现在你来帮帮小哼吧。小哈给小哼加密过的一串数是“6 3 1 7 5 8 9 2 4”。解密后小哈的QQ号应该是“6 1 5 9 4 7 2 8 3”。

输入格式:

只有2行

第1行有一个整数n （1<=n<=100000）

第2行有n个整数为加密过的QQ号，每个整数之间用空格隔开。每个整数在1~9之间。

输出格式:

只有一行，输出解密后的QQ号。

限制:

每个测试点1秒

50%的数据1<=n<=10000

100%的数据1<=n<=100000

样例 1 :

输入:
9
6 3 1 7 5 8 9 2 4

输出:
6 1 5 9 4 7 2 8 3
```

### 代码

```c++
#include <iostream>
using namespace std;

struct queue{
	int data[200000];
	int head;
	int tail;
};

int main()
{
	queue q;
	int n;
	cin>>n;
	q.head = 0;
	q.tail = 0;
	for(int i=0;i<n;i++){
		cin>>q.data[q.tail];
		q.tail++;
	}
	while(q.head<q.tail){
        //打印队列头
		cout<<q.data[q.head]<<" ";
        //删除队列头
		q.head++;
        //将第下一个放入队列尾
		q.data[q.tail] = q.data[q.head];
        //队列尾右移
		q.tail++;
        //删除队列头
		q.head++;
	}
	return 0;
}
```
